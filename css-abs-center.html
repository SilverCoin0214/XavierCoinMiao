各类居中方案

    水平居中

      1. 行内元素

    • 1.1 行内元素:父元素设置text-align:center就可以
      • abc
    • 1.2 行内块元素: 父元素设置成text-align:center,子元素设置为inline-block.有基线用父元素的font-size:0去除

      2. 块级元素

    • 2.1 子元素设置成margin: 0 auto;
    • 2.2 使用table+margin: display:table类似于block元素,但宽度为内容宽
      • table
    • 2.3 使用absolute+transform: 先父元素相对定位,子元素绝对定位, 右移子元素,距离为父元素宽度的50%, 再左移子元素,距离为自身宽度的50%
      • abs+trans
    • 2.4 使用flex+justify-content:center
      • flex+justify
    • 2.5 使用flex+margin: 父元素设置flex,子元素设置margin:0 auto

      3. 多块级元素

    • 3.1 使用flex+ justify-content
    • 3.2 使用inline-blcok: 父元素设置text-align:center

    垂直居中

      1. 单行内联元素

    • 1.1 使用line-heiht=height
      • height

      2. 多行内联元素

    • 2.1 使用flex里的flex-direction:column

      • Lorem ipsum dolor sit amet, consectetur adipisicing elit.

    • 2.2 使用flex里的align-items:center

      • Lorem ipsum dolor sit amet, consectetur adipisicing elit.

      3. 块级元素

    • 3.1 使用absolute+负margin: 绝对定位到顶部的50%, 并设置margin-top向上偏移元素自身高度的一半
    • 3.2 使用absolute+transform: 当元素高度未知时,可以使用transform:translateY来反向偏移50%实现
      • transform:translateY
    • 3.3 使用flex+align-items:center. 直接使子元素垂直居中
      • align-items
    • 3.4 使用table-cell+ vertical-align: 使父元素变成表格单元格, 设置vertical-algin属性, 使子元素垂直居中
      • vertical-algin:middle
